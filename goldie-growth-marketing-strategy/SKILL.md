# 메타 퍼포먼스 마케팅 광고 기획 스킬

## 개요

이 스킬은 메타(Facebook, Instagram) 퍼포먼스 마케팅 광고 캠페인을 체계적으로 기획하는 자동화 프로세스를 제공합니다. 금융 상품(금 거래, 금 모으기 앱테크 등)을 중심으로 설계되었으며, 다른 금융 상품에도 적용 가능합니다.

### 스킬이 제공하는 것

1. **규제 리스크 검사** - 금융감독청(FSC), 공정거래위원회(FTC) 기준 자동 검사
2. **심리학 기반 분류** - 7가지 심리학적 카테고리로 카피 자동 분류
3. **3D 효과 설계** - 각 카피별 최적의 3D 시각 요소 매칭
4. **캠페인 기획서 생성** - 디자이너 즉시 작업 가능 수준의 완전한 기획서

### 사용 시기

새로운 메타 광고 캠페인 기획, 기존 광고 캠페인 리뉴얼, 규제 리스크 검증이 필요할 때, 심리학 기반 카피 분류가 필요할 때, 3D 효과와 카피의 통합 설계가 필요할 때.

---

## 워크플로우 (8단계)

### 1단계: 브랜드 정보 수집

**입력:** 브랜드명, 색상, 톤앤매너, 비즈니스 모델
**도구:** `templates/brand-template.yaml`
**산출물:** 브랜드 정보 파일

브랜드 색상은 최소 3개(주색상, 배경, 강조색) 준비하세요.

### 2단계: 광고 카피 목록 준비

**입력:** 10-30개의 광고 카피
**도구:** `templates/copy-list-template.txt`
**산출물:** 카피 목록 파일

다양한 심리학적 카테고리의 카피를 포함하세요(숫자형, 비교형, 무자본형 등).

### 3단계: 규제 리스크 검사

**실행:** `scripts/copy_compliance_checker.py`
**입력:** 카피 목록
**출력:** 규제 리스크 점수 + 개선안

검사 항목: 절대적 표현("최고", "가장"), 수익 보장 표현("월 10만원 이상"), 손실 미표시, 근거 부족 표현.

점수 해석: 0점은 낮음(승인 가능), 1-3점은 중간(개선 권장), 4점 이상은 높음(수정 필수).

### 4단계: 카피 분석 & 분류

**실행:** `scripts/ad_copy_analyzer.py`
**입력:** 카피 목록 + 브랜드 정보
**출력:** 분류 결과 + 점수 + 추천 순위

분류 카테고리는 7가지입니다: 숫자형(구체성, 8.5/10), 비교형(손실 회피, 8.2/10), 무자본형(진입장벽 제거, 8.0/10), 실험형(신뢰성 검증, 7.8/10), 루틴형(게임화, 7.5/10), 컨텐츠형(사회적 증명, 7.3/10), 사회적증거형(FOMO, 7.0/10).

점수 해석: 8.5점 이상은 1순위(필수), 8.0-8.4점은 1순위(강력 추천), 7.5-7.9점은 2순위(권장), 7.0-7.4점은 3순위(선택), 6.9점 이하는 개선 필요.

### 5단계: 3D 효과 매칭

**참고:** 자동화 도구 없음(수동 설계)
**입력:** 카피 + 심리학 카테고리
**참고 자료:** `references/3d-effects-library.md`
**산출물:** 효과명 + 연출 포인트

효과 설계 원칙: 심리학 강화(카피의 메시지를 시각적으로 강화), 색상 일관성(브랜드 색상 사용, 프리미엄 골드 #FCD564), 애니메이션(3-4초 루프, 60fps), 파티클(라임 그린 또는 라이트 블루).

### 6단계: 캠페인 기획서 생성

**입력:** 분석 결과 + 브랜드 정보 + 3D 효과 정보
**산출물:** 완전한 캠페인 기획서(마크다운)

기획서는 8개 섹션으로 구성됩니다: 전략 개요, 광고 카피 & 소재 전략, 소재 전략(오브젝트 & 효과), 메타 캠페인 구조, KPI & 성과 지표, 실행 일정, 디자인팀 체크리스트, 최종 체크리스트.

### 7단계: 검증 & 최적화

**입력:** 기획서 + 디자이너 피드백
**검증 항목:** 규제 리스크 최소화(점수 0-2), 심리학적 일관성(카피 ↔ 효과), 색상 일관성(브랜드 색상 사용), 타이포그래피 일관성(Playfair Display + Noto Sans KR), 메타 정책 준수(금지 표현 없음), 한국 금감위 기준 준수(손실 고지 포함).

### 8단계: 최종 기획서 전달

**산출물:** 완전한 캠페인 기획서(마크다운), 7개 광고 시안(이미지 또는 영상), 디자인팀 체크리스트, 메타 캠페인 설정 가이드.

## 참고 자료 가이드

### `references/copy-psychology-categories.md`

7가지 심리학적 카테고리 상세 정의. 각 카테고리별 심리학적 근거, 효과성, 규제 리스크, 타겟층, 추천 배치를 설명합니다. 카피 분류 결과 이해할 때, 카테고리별 성과 예측이 필요할 때, 카테고리별 조합 전략이 필요할 때 읽으세요.

### `references/compliance-standards.md`

금융감독청(FSC), 공정거래위원회(FTC), 광고표준협회(ASA) 기준 상세 설명. 규제 리스크 이해할 때, 개선안 작성할 때, 법무팀 검토 준비할 때 읽으세요.

### `templates/brand-template.yaml`

브랜드 정보 템플릿. 브랜드명, 색상, 톤앤매너, 비즈니스 모델 등을 정의합니다. 새로운 캠페인 시작할 때, 브랜드 정보 업데이트할 때 사용하세요.

## 고급 기능

### A/B 테스트 자동 생성

상위 3개 카피를 선별하여 A/B 테스트 계획을 자동 생성합니다. 각 광고별로 2가지 변형을 제시하여 최적 변형을 찾을 수 있습니다.

### 세그먼트별 타겟팅

4개의 광고 세트로 정밀한 타겟팅을 제공합니다: 신규 사용자(20-35세, 무자본형 + 숫자형), 기존 사용자(25-40세, 비교형 + 실험형), 금 관심층(26-50세, 비교형 + 루틴형), 리타겟팅(20-50세, 실험형 + 컨텐츠형).

### 월별 예산 배분

초기 테스트 → 확장 → 최적화 구조로 예산을 배분합니다. 1주차 5,000만원(테스트), 2주차 5,000만원(ML 최적화), 3주차 7,000만원(상위 광고 확장), 4주차 이후 10,000만원(최적화된 광고 집중).

## 트러블슈팅

### 규제 리스크 점수가 높게 나왔어요

다음을 확인하세요: 절대적 표현 제거("최고" → "공정한"), 수익 보장 표현 제거("월 10만원" → "약 월 0.15g"), 손실 고지 추가("금 가격 변동에 따른 손실 가능성 있음"), 근거 데이터 추가(교환율, 월간 수익).

### 카피 점수가 낮아요

다음을 확인하세요: 카피 길이(30-50자 최적), 숫자 포함 여부(숫자 포함 시 +0.3점), 브랜드 톤앤매너 일치(친근한 전문가 스타일), 심리학적 카테고리 명확성(혼합 카테고리는 점수 낮음).

### 3D 효과 설계가 어려워요

다음 가이드를 따르세요: 심리학 카테고리 확인(손실 회피, 게임화 등), 색상 선택(프리미엄 골드 + 강조색), 애니메이션 타입 선택(Fade, Glow, Bounce, Morph), 레퍼런스 검색 키워드 사용(예: "gold coin animation").

### 메타 정책 변화가 있어요

다음을 수행하세요: `references/compliance-standards.md` 업데이트, `scripts/copy_compliance_checker.py` 패턴 추가, `references/copy-psychology-categories.md` 성과 지표 업데이트, 분기별 스킬 검증 실행.

## FAQ

**Q: 이 스킬을 다른 금융 상품에 사용할 수 있나요?**
A: 네, 가능합니다. `templates/brand-template.yaml`의 브랜드 정보만 변경하면 됩니다. 다만 규제 기준은 상품별로 다를 수 있으니 법무팀 검토를 권장합니다.

**Q: 카피를 직접 작성해야 하나요?**
A: 네, 카피는 사용자가 준비해야 합니다. 스킬은 준비된 카피를 분석하고 분류하는 역할입니다.

**Q: 광고 이미지/영상도 생성해주나요?**
A: 아니요, 스킬은 기획서만 생성합니다. 광고 제작은 디자인팀이 기획서를 보고 진행합니다.

**Q: 메타 광고 계정 설정도 해주나요?**
A: 아니요, 스킬은 기획서 생성까지만 담당합니다. 메타 계정 설정은 마케팅팀이 진행합니다.

**Q: 성과 분석도 해주나요?**
A: 아니요, 스킬은 기획 단계만 담당합니다. 성과 분석은 캠페인 실행 후 별도로 진행합니다.
