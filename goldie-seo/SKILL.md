# Goldie SEO Strategy

금골디 웹/앱/블로그의 검색 최적화 전략을 수립하고 실행한다.

## 역할

1. **키워드 리서치 & 매핑** — 검색 수요 분석, 키워드 우선순위, 퍼널별 매핑
2. **온페이지 SEO** — 제목/메타/H태그/내부링크 최적화
3. **기술 SEO** — Core Web Vitals, 크롤링, 인덱싱, 모바일
4. **스키마 마크업** — JSON-LD 구조화 데이터 생성/검증
5. **AI 검색 최적화 (AEO/GEO)** — ChatGPT, Perplexity 등 AI 검색 대응
6. **콘텐츠 클러스터** — 필러-클러스터 전략, 토픽 오소리티
7. **경쟁사 SEO 분석** — 키워드 갭, 백링크, 콘텐츠 갭

## 필수 연동

- **goldie-marketer**: SEO 결과를 콘텐츠 제작에 반영
- **goldie-bible**: 금 도메인 키워드 정확성 검증
- **goldie-researcher**: 경쟁사/시장 SEO 동향 조사

## 1. 키워드 리서치 프레임워크

### 키워드 분류 체계

| 의도 | 설명 | 금골디 예시 | 콘텐츠 유형 |
|------|------|-----------|-----------|
| 정보형 (Informational) | 알고 싶다 | "금시세 보는 법", "금 투자 방법" | 블로그, FAQ |
| 탐색형 (Navigational) | 찾고 싶다 | "금골디 앱", "금골디 매입가" | 서비스 페이지 |
| 상업형 (Commercial) | 비교하고 싶다 | "금 거래 앱 비교", "금은방 vs 앱" | 비교 콘텐츠 |
| 거래형 (Transactional) | 하고 싶다 | "금 팔기", "금 매입 신청" | 랜딩페이지 |

### 키워드 매핑 템플릿

```
타겟 키워드: [메인 키워드]
검색량: [월간 추정]
난이도: [상/중/하]
검색 의도: [정보/탐색/상업/거래]
현재 순위: [있으면 기재]
타겟 페이지: [URL]
서브 키워드: [2-3개]
롱테일: [3-5개]
```

### 금골디 핵심 키워드 클러스터

| 클러스터 | 필러 키워드 | 클러스터 키워드 |
|---------|-----------|--------------|
| 금 시세 | 금시세, 오늘의금시세 | 금시세 전망, 금시세 그래프, 금한돈시세, 실시간 금시세 |
| 금 거래 | 금거래, 금매입 | 금팔기, 금매입가, 금거래 방법, 금거래 수수료 |
| 금 투자 | 금투자, 금테크 | 소액 금투자, 금투자 방법, 금 ETF vs 실물금 |
| 금 종류 | 순금, 18K, 14K | 금반지 시세, 돌반지 시세, 골드바 가격 |
| 금 상식 | 금 상식 | 금 순도, 금 공임, 금 감정, 금 세금 |

## 2. 온페이지 SEO 체크리스트

### 필수 요소

| 요소 | 규칙 | 금골디 적용 |
|------|------|-----------|
| Title 태그 | 핵심 키워드 포함, 55자 이내 | "[키워드] - 금골디" 패턴 |
| Meta Description | 핵심 키워드 + CTA, 155자 이내 | "금골디에서 [키워드] 확인하세요" |
| H1 | 페이지당 1개, 핵심 키워드 포함 | 제목 = H1 |
| H2-H3 | 서브 키워드 자연 삽입 | 논리적 위계 |
| URL 구조 | 짧고 키워드 포함, 한글 가능 | `/blog/금시세-보는법` |
| 이미지 ALT | 키워드 포함 설명 | "금골디 앱 실시간 금시세 화면" |
| 내부 링크 | 관련 콘텐츠 3-5개 연결 | 앵커텍스트에 키워드 |
| 캐노니컬 | 중복 페이지 방지 | 모든 페이지에 설정 |

### 콘텐츠 길이 가이드

| 콘텐츠 유형 | 권장 길이 | 이유 |
|-----------|---------|------|
| 인블로그 일반 | 1,500-2,500자 | 네이버 검색 상위 노출 최적 |
| 인블로그 가이드 | 3,000-5,000자 | 포괄적 정보 제공, 체류 시간 증가 |
| 파워콘텐츠 랜딩 | 700-1,500자 | 규정 최소 700자 + 전환 집중 |
| FAQ 페이지 | 질문당 100-300자 | 간결한 답변, 리치 스니펫 노출 |

## 3. 기술 SEO

### Core Web Vitals 기준

| 지표 | 양호 | 개선 필요 | 불량 | 측정 |
|------|------|---------|------|------|
| LCP (최대 콘텐츠 페인트) | ≤2.5s | ≤4.0s | >4.0s | 가장 큰 콘텐츠 요소 로드 시간 |
| INP (다음 페인트 상호작용) | ≤200ms | ≤500ms | >500ms | 사용자 상호작용 응답 시간 |
| CLS (누적 레이아웃 이동) | ≤0.1 | ≤0.25 | >0.25 | 시각적 안정성 |

### 크롤링 & 인덱싱 체크

- [ ] robots.txt가 필요한 페이지만 차단하는가?
- [ ] XML 사이트맵이 최신 상태인가?
- [ ] 404 에러 페이지가 유용한 안내를 제공하는가?
- [ ] 리다이렉트 체인이 없는가? (최대 1홉)
- [ ] 모바일 반응형이 완전한가?
- [ ] HTTPS가 전체 적용되었는가?
- [ ] 페이지 로드 속도가 3초 이내인가?

### AI 크롤러 대응

2025년부터 GPTBot, ClaudeBot, PerplexityBot 등 AI 크롤러 트래픽이 급증.

| 크롤러 | User-Agent | 대응 |
|--------|-----------|------|
| GPTBot | GPTBot | 허용 (AI 검색 노출 위해) |
| ClaudeBot | ClaudeBot | 허용 |
| PerplexityBot | PerplexityBot | 허용 |
| Google-Extended | Google-Extended | 허용 (Gemini 학습 포함) |

> 금골디는 AI 검색 노출이 중요하므로 주요 AI 크롤러를 **허용**한다.

## 4. 스키마 마크업 (구조화 데이터)

### 금골디 필수 스키마

#### Organization
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "금골디",
  "url": "https://goldie.co.kr",
  "logo": "[로고 URL]",
  "description": "금 거래 중개 플랫폼",
  "sameAs": [
    "https://www.instagram.com/goldie_official",
    "[기타 소셜 URL]"
  ]
}
```

#### FAQPage (블로그/서비스 FAQ)
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "금골디에서 금을 팔려면 어떻게 하나요?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "앱에서 MY귀금속을 등록하고 거래 신청을 하면, 입찰 기반으로 최적의 매입가를 받을 수 있습니다."
      }
    }
  ]
}
```

#### Article (블로그 글)
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "[제목]",
  "author": {"@type": "Organization", "name": "금골디"},
  "datePublished": "[발행일]",
  "dateModified": "[수정일]",
  "description": "[메타 디스크립션]"
}
```

#### SoftwareApplication (앱)
```json
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "금골디",
  "operatingSystem": "Android, iOS",
  "applicationCategory": "FinanceApplication",
  "offers": {"@type": "Offer", "price": "0"}
}
```

### 스키마 검증 체크리스트
- [ ] JSON-LD 문법이 유효한가?
- [ ] Google Rich Results Test 통과하는가?
- [ ] 페이지 콘텐츠와 스키마 데이터가 일치하는가?
- [ ] 필수 속성이 모두 포함되었는가?

## 5. AI 검색 최적화 (AEO / GEO)

### 3축 전략

| 축 | 설명 | 금골디 적용 |
|----|------|-----------|
| **AEO** (Answer Engine Optimization) | AI가 답변으로 인용할 수 있도록 구조화 | FAQ 형식, 질문-답변 페어, 정의형 문단 |
| **GEO** (Generative Engine Optimization) | AI 생성 답변에 출처로 포함되도록 | 통계/수치 포함, 출처 명시, 독자적 데이터 제공 |
| **LLMO** (LLM Optimization) | LLM 학습 데이터에 포함되도록 | 크롤러 허용, 명확한 HTML 구조, 메타데이터 |

### AI 인용 최적화 원칙

1. **직접 답변 제공**: "금시세란?" → 첫 문단에 명확한 정의
2. **수치와 출처**: "국내 소비자 평균 순금 보유량 약 13.3돈 (2023, Bible)"
3. **독자적 데이터**: 금골디만의 거래 데이터, 매입가 비교 등 고유 정보
4. **구조화된 형식**: 표, 목록, 단계별 설명 → AI가 파싱하기 쉬움
5. **E-E-A-T 강화**: 작성자 전문성, 실제 거래 경험, 출처 인용

### 모니터링

| 도구 | 용도 |
|------|------|
| Google Search Console | 검색 유입, 키워드 순위, 클릭률 |
| Ahrefs / SEMrush | 키워드 추적, 백링크, 경쟁사 분석 |
| ChatGPT / Perplexity 직접 검색 | "금 거래 앱 추천" 등으로 노출 여부 확인 |
| AnswerThePublic | 질문형 키워드 발굴 |

## 6. 콘텐츠 클러스터 전략

### 필러-클러스터 모델

```
              [필러 페이지]
            "금 거래 완벽 가이드"
           /      |       \
    [클러스터]  [클러스터]  [클러스터]
   "금시세     "금팔기    "금 종류별
    보는법"     방법"      차이"
      |          |          |
  [서브]      [서브]     [서브]
 "실시간     "금은방 vs  "18K vs
  금시세"     앱 비교"   24K"
```

### 내부 링크 전략

| 규칙 | 설명 |
|------|------|
| 필러 → 클러스터 | 필러 페이지에서 모든 클러스터로 링크 |
| 클러스터 → 필러 | 모든 클러스터에서 필러로 역링크 |
| 클러스터 ↔ 클러스터 | 관련 클러스터 간 상호 링크 |
| 앵커 텍스트 | 키워드 포함 자연스러운 문구 |
| 신규 콘텐츠 | 발행 시 기존 관련 콘텐츠에 링크 추가 |

### 토픽 오소리티 빌딩

1. **1차 (현재)**: 금 시세/거래 관련 핵심 키워드 20개 콘텐츠 확보
2. **2차**: 금 투자/자산관리 확장 키워드 30개 추가
3. **3차**: 귀금속 전반 (은, 백금) + 재테크 연계 콘텐츠

## 7. 경쟁사 SEO 분석

### 분석 대상

| 경쟁사 | 주요 채널 | SEO 강점 |
|--------|---------|---------|
| 한국금거래소 | 웹사이트, 블로그 | 브랜드 키워드 장악, 도메인 오소리티 |
| 금방금방 | 블로그, 유튜브 | 콘텐츠 볼륨, 교육 콘텐츠 |
| 업스토어 | 웹, 블로그 | 가격 비교 키워드 |

### 분석 항목

1. **키워드 갭**: 경쟁사는 잡지만 금골디는 없는 키워드
2. **콘텐츠 갭**: 경쟁사가 다루지 않는 주제 (금골디 기회)
3. **백링크 프로필**: 경쟁사 백링크 출처 → 동일 출처 확보 가능성
4. **SERP 특성**: 리치 스니펫, People Also Ask 점유 현황

### 경쟁 우위 SEO 전략

- **정보 비대칭 해소 콘텐츠**: 경쟁사가 다루지 않는 "매입가 차이 구조" 설명
- **사용자 데이터 기반**: 금골디만의 거래 데이터/통계 활용
- **비교 콘텐츠**: "[경쟁사] vs 금골디" 대신 "금 거래 방법 비교" (직접 비교 금지 규칙 준수)

## 8. 플랫폼별 SEO

### 네이버 SEO (인블로그)

| 요소 | 네이버 특화 규칙 |
|------|---------------|
| C-Rank | 블로그 주제 일관성 유지 (금/재테크 집중) |
| D.I.A. | 문서 품질 — 독자적 내용, 적절한 길이, 멀티미디어 |
| 키워드 | 네이버 키워드 도구 + 자동완성 활용 |
| 스마트블록 | FAQ, 가이드 형식 → 스마트블록 노출 |

### 인스타그램 SEO

goldie-marketer 에이전트의 인스타 SEO 규칙을 따르되, 추가로:
- **프로필 SEO**: 이름 필드에 "금골디 | 금거래" 키워드
- **릴스 캡션**: 첫 3줄에 키워드 집중
- **Alt Text**: 모든 이미지에 키워드 포함 대체 텍스트

### 앱스토어 최적화 (ASO)

| 요소 | iOS (App Store) | Android (Play Store) |
|------|----------------|---------------------|
| 앱 이름 | 30자, 핵심 키워드 포함 | 30자, 핵심 키워드 포함 |
| 부제 | 30자 | N/A |
| 짧은 설명 | N/A | 80자 |
| 키워드 필드 | 100자 (쉼표 구분) | N/A (설명에서 추출) |
| 스크린샷 | 첫 2장이 핵심 | 첫 2장이 핵심 |
